# AcumaticaRESTAPIClientForCSharp
Acumatica REST API Client for C#

The solution allows to easily start developing REST API based integration with Acumatica ERP. 

The solution contains following projects:
1. Acumatica.RESTClient - the project contains Acumatica REST API client based on RESTSharp. The project also contains base classes required for endpoint definitions.
2. Acumatica REST API Console Application - the project contains examples of API calls to Acumatica ERP
4. Acumatica.Default_18.200.001 - the project contains REST API Endpoint client for Default Endpoint version 18.200.001
4. Acumatica.Default_20.200.001 - the project contains REST API Endpoint client for Default Endpoint version 20.200.001
4. Acumatica.DeviceHub_19.200.001 - the project contains REST API Endpoint client for DeviceHub Endpoint version 19.200.001
5. Acumatica.eCommerce_20.200.001 - the project contains REST API Endpoint client for eCommerce Endpoint version 20.200.001
5. Acumatica.ISVCB_19.1 - the project contains REST API Endpoint client for ISVCB Endpoint version 19.1
5. Acumatica.Manufacturing_18.200.001 - the project contains REST API Endpoint client for Manufacturing Endpoint version 18.200.001
5. Acumatica.Manufacturing_20.200.001 - the project contains REST API Endpoint client for Manufacturing Endpoint version 20.200.001
6. SOAPLikeWrapperForREST - the project contains wrapper that can be used to migrate a code that has originally been written for SOAP API to REST API
